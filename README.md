# WhatsApp Photo Fixer

Herramienta en Python con PyQt para restaurar la fecha de captura de imágenes de WhatsApp usando el nombre del archivo.

## Funcionalidades
- Selección de carpeta
- Procesamiento automático de imágenes
- Restauración de EXIF

## Uso
1. Ejecutar main.py
2. Seleccionar carpeta
3. Procesar

## Requisitos
pip install -r requirements.txt
